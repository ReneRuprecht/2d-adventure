# 2D-Adventure

# How to Use
1. Clone repository
2. cd into directory
3. Run commands
4. go to localhost:4567
5. enjoy

## Commands
 
    Run docker build -f docker-depl . -t rma-2d-adventure
    Run docker run -p 4567:80 --name rma-2d-adventure rma-2d-adventure
